Projet de Corentin GRALL et Sullyvan YONNET

Les techonologies utilisées:

	- React
	- MongoDB
	- Tomcat
	- JDBC
	- MariaDB
	- React Bootstrap
	- JSON
	
Outis a installer:

	- npm install react-responsive-carousel --save
	- npm install --save bootstrap
	
Outis a importer

	- mariadb-java-client-2.4.1-sources.jar
	- mysql-connector-java-8.0.13.jar
	- mysql-connector-java_8.0.15-1ubuntu18.04_all.deb
	- jackson-databind-2.8.2.jar
	- jackson-core-2.8.2.jar
	- jackson-annotations-2.8.0.jar
Les fonctionnalitées :

	Sans être connecté
		- Possibilité de connexion.
		- Possibilité de s'inscrire.
		- Controle de qualité des informations saisies lors de l'inscription et affichage du message d'erreurs adapter.
		- Affichage des voyage. Avec comme informations Une image, un titre, une date de debut et de fin, une description,un prix. 
		- Affichage d'un voyage précis avec un boutton réservé grisé. les informations afficher sont les mêmes que précédament avec en plus plusieurs photos au lieu de une.
		- Affichage des commentaires lié à un voyage sans être connecté. Les informations sont le nom de l'utilisateur qui a commenté et son commentaire
	
	En etant connecté
		- Affichage du nombre d'élément dans le panier à coté de l'icon panier.
		- Possibilité de retourner à l'accueil avec le mot "ACCUEIL" dans le header.
		- Possibilité de se déconnecter.
		- Possibilité de reservé un voyage grace au bouton "réservé" qui n'est plus grisé dans un page d'un voyage.
		- Possibilité d'afficher son panier avec l'icon du panier dans le header.
		- Possibilité de reglé le panier dans la page panier.
